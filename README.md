# Bit - Self-Hosted Autonomous Personal AI Assistant

## Overview

Bit is a self-hosted autonomous personal AI assistant that I designed to help me manage my personal life and balance my academic responsibilities. I wanted it to run off my laptop or dorm PC so that it's completely free.

I envisioned Bit as a friendly assistant that learns from my habits and preferences and behaves as close to a remote executive assistant as possible.

The name "Bit" was chosen for the project in reference to a early modular AI agent prototype I designed my Sophomore year of college which had named itself "Bit".

## Capabilities

*Bit is able to perform the following tasks:*
1. Read Canvas assignments and predict time to complete
2. Create a homework schedule based on Canvas assignments
3. Read Google Calendar events and send reminders
4. Read important emails and send reminders
5. Operate asynchronously like a real executive assistant
6. Behave like a friendly, executive assistant for personal life management
